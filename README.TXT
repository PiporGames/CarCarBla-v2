CarCarBla-v2 es un proyecto de C++ que consiste en un programa que calcula la mejor ruta posible entre una serie de destinos posibles.
El proyecto utiliza diversos algoritmos como Dijkstra o Floyd para obtener la mejor ruta entre dos puntos de un grafo.
El objetivo del proyecto es, evidentemente, familiarizarse y comprender estos tipos de algoritmos y saber utilizarlos para un propósito específico.
Existe documentación disponible para el proyecto en el repositorio.
Este proyecto ha sido creado para la asignatura de Análisis Y Diseño De Algoritmos del Grado de Ingeniería en Ingeniería del Software de la UEx año 2023/2024.  

Creado por José Manuel de Torres Domínguez.  

En este repositorio se encuentra el proyecto de Visual Studio 2022 (v17) C++.
Para su ejecución es recomendable utilizar las configuraciones de ejecución ya diseñadas.
Es necesario tener instalados el paquete de Visual Studio 2022 Community para ejecutar, o por lo menos las herramientas necesarias para compilar el proyecto por separado.
